概念学习

做机器学习还是需要一定的数学基础的，需要从公式中去获得实现算法的原理。
在逻辑回归模型中，以下公式发挥着巨大的作用：

公式：

1. 归一化公式

用于将特征值缩放到0和1之间的归一化公式：

[ \text{norm_X} = 1 - \frac{(\text{maxs} - X)}{\text{rng}} ]

其中，maxs 是每列特征的最大值，mins 是每列特征的最小值，rng 是范围 (maxs - mins)。

2. 逻辑（Sigmoid）函数

逻辑回归模型中使用的sigmoid函数，用于将任意实数映射到0和1之间，表示为概率：

[ \sigma(x) = \frac{1}{1 + e^{-x}} ]

在上下文中，具体形式为：

[ \text{logistic_func}(\beta, X) = \frac{1}{1 + \exp(-X \beta^T)} ]

3. 对数似然成本函数

逻辑回归中使用的对数似然成本函数，用于计算模型的预测损失：

[ J(\beta) = -\frac{1}{n} \sum_{i=1}^{n} \left[ y^{(i)} \log(\sigma(x^{(i)})) + (1 - y^{(i)}) \log(1 - \sigma(x^{(i)})) \right] ]

其中，( \sigma(x^{(i)}) ) 是模型预测的概率，( y^{(i)} ) 是实际标签。

4. 梯度计算公式

用于计算成本函数梯度的公式，用于在梯度下降中更新参数：

[ \nabla J(\beta) = \frac{1}{n} \sum_{i=1}^{n} (\sigma(x^{(i)}) - y^{(i)}) x^{(i)} ]

5. 梯度下降更新规则

在梯度下降算法中使用的参数更新规则：

[ \beta := \beta - \alpha \nabla J(\beta) ]

其中，( \alpha ) 是学习率，用于控制步长大小。

6. 决策边界公式

在二维空间中，用于绘制决策边界的直线方程，当使用两个特征时：

[ x_2 = -\frac{(\beta_0 + \beta_1 x_1)}{\beta_2} ]

其中，( \beta_0, \beta_1, \beta_2 ) 是逻辑回归模型的参数，( x_1 ) 和 ( x_2 ) 是特征。

这些数学公式是机器学习模型，特别是逻辑回归在理论和实践中的基础。它们有助于理解模型是如何从数学角度构建和优化的。

EDA

EDA（Exploratory Data Analysis），全名数据探索性分析，是通过了解数据集，了解变量间的相互关系以及变量与预测值之间的关系，从而帮助我们后期更好地进行特征工程和建立模型，是数据挖掘中十分重要的一步。
